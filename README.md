# NutriSync AI 🥗

## 📌 Problem Statement

Many people struggle to maintain a healthy diet because they don't know what food is best for their body, lifestyle, or fitness goals.

## 💡 Solution

NutriSync AI is an AI-powered nutrition assistant that helps users get personalized diet recommendations based on their health goals, preferences, and daily habits.

## 🚀 Features

* AI based diet recommendations
* Personalized meal planning
* Nutrition tracking
* User friendly interface
* Smart food suggestions

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js / Python 
* Database: MongoDB / MySQL 
* AI Tools: OpenAI
